# Anesthesia Companion

**Your Buddy Through Every Anesthesia Rotation.**

A professional, premium medical education platform for medical students and junior doctors during anesthesia rotation. Built with a dark-mode, glassmorphism aesthetic inspired by Apple, Notion, and Linear.

## Tech Stack

- **Framework**: TanStack Start (React 19 + file-based routing)
- **Styling**: Tailwind CSS v4 + custom CSS (dark mode glassmorphism)
- **Language**: TypeScript
- **Deployment**: Netlify

## Features

16 comprehensive sections:

| Section | Description |
|---------|-------------|
| **Home** | Hero, quick access cards, global navigation |
| **Drugs** | 15+ drug cards with mechanism, doses, pearls, contraindications |
| **Fluids** | Crystalloids, colloids, blood products, maintenance calculator |
| **Airway** | LEMON assessment, DAS algorithm, BMV technique |
| **Intubation** | Step-by-step RSI guide with interactive checklist |
| **Airway Anatomy** | Interactive anatomical structure explorer |
| **Inhalational** | Volatile agents with MAC/blood:gas comparison table |
| **Muscle Relaxants** | NMBs with TOF monitoring guide, comparison table |
| **Hemodynamics** | MAP calculator, hypotension/hypertension management |
| **Machine** | Interactive anesthesia machine screen with clickable parameters |
| **Emergencies** | 11 emergency protocols with algorithms, drugs, red flags |
| **Clinical Pearls** | 18+ high-yield pearls with categories and search |
| **Cases** | 3 interactive clinical cases with self-assessment MCQs |
| **Notes** | Personal notes with CRUD, saved to localStorage |
| **Favorites** | Saved drugs, pearls, and quick references |
| **Search** | Global search across all sections |

## Running Locally

```bash
npm install
npm run dev
```

App runs at [http://localhost:3000](http://localhost:3000).
